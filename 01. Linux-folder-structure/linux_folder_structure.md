
# 🐧 Linux Folder Structure

```
/
├── 🏠 home        # User personal files
│   └── 👤 imran   # Example user folder
├── 👑 root        # Root (admin) user home
├── ⚒️ bin         # Basic commands/programs
├── 🛠️ sbin        # System admin commands
├── ⚙️ etc         # Configuration files
├── 📂 var         # Variable data (logs, mails, etc.)
│   └── 📜 log     # Log files
├── 🗑️ tmp         # Temporary files
├── 📦 usr         # User installed programs/files
│   └── ⚒️ bin     # Extra commands
├── 📚 lib         # Shared libraries
├── 🔑 boot        # Boot loader files
├── 💽 dev         # Device files
├── 📊 proc        # Processes & system info
├── 📌 mnt         # Temporary mount points
└── 💿 media       # External devices (USB, CD, etc.)
```
